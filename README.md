# Foreword
This repository offers a technical reference of the Calcolatrice Elettronica Pisana (CEP) computer. 

The information here is mainly targeted at the development of an emulator for the machine. 

The main source for this guide are scans of paper's from the 1960 regarding the computer, offered freely online at the following [link](http://csce.isti.cnr.it/index.php/ii-serie/pubblicazioni-ii).

## Book structure
Documents will be divided in two groups:
- those aimed at describing one component of the CEP computer. 

- those aimed at offering general information regarding computer from the era.

This are stored inside the folder **"documents"**. The [index](documents/0_index) has a list of all documents for convenience sake.

## Terminology
The describtions will keep for coherence sake the original terminology used in the 1960s. Names of components will try to be in english, but if there wasn't available an english paper showing the name,then it will be used its original italian name. 

The [glossary](documents/0_glossary) will keep a map between old terms, offering a brief description and the equivalent modern-day name.
