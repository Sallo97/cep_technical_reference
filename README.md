# Foreword
This repository offers a technical reference of the Calcolatrice Elettronica Pisana (CEP) computer. 

The information here is mainly targeted at the development of an emulator for the machine. 

The main source for this guide are scans of paper's from the 1960s regarding the computer, offered freely online at the following [link](http://csce.isti.cnr.it/index.php/ii-serie/pubblicazioni-ii).

Sources are cited across the book, all contained in the the [reference](./documents/reference) section. 

## Book structure
The book is divided in chapters, each of which aims to describe completely a component of the CEP's architecture. A Chapter is composed of multiple documents, some provide description of parts of the components, while other will focus on more general information regarding the era.

These are stored inside the folder **"documents"**. 

It is recommended to refer to the [index](documents/0_index).

## Terminology
The descriptions will stick to the original terminology used in the 1960s. Names of components will try to be in english, but if not available,then it will be used its italian name. 

The [glossary](documents/0_glossary) will provide for each term a brief description and if it exists will specify its equivalent modern-day name.
