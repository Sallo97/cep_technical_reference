# Symbolic CEP to Binary CEP Translator (TSBC)
TSBC stands for "*FORTRAN Origin to FORTRAN CEP Translator*" (from the italian "*Traduttore FORTRAN ORIGIN a FORTRAN CEP*"). It has the responsabilities of translating routines written in LPSC into binary, ensuring the correctness of the given code. If an error is detected, the process stops and the compiler returns some error messages. Note that it does not print only the ifrst error it encounters, but all of them. [[7](../helper_resources/references.md)]

In case of a successful compilation, if requested it can print useful debug information which can be used by the assembler. various message types can be written, each containing different informations. Some information can be written into a punched tape, primarily to pass to the assembler, or printed into the teleprinter.[[7](../helper_resources/references.md)] 

- The original definition of each routine. Additionally it can specify where it came from.

- Each pseudoinstruction sequence and their associated macroprogram.

- The list of all used subprograms' names.

- The generated binary code.

- The printing (through the teleprinter) or punching (through the tape) of the original program side by side to its octal representation.

- Printing or punching of all the tables generated by TSBC during the translation process.
